<objective>
Fix broken slide-in/slide-out transitions on PrivacyNotice and TrademarkNotice bottom-sheet panels.

Purpose: Both panels are currently conditionally mounted with `{showX && ...}` guards, which destroys the component before exit animation can play. The animation approach (`transition-[inset]`) is also ineffective cross-browser for this use case. This fix makes the panels always-mounted and switches to the proven `transition-transform` + `translate-y` approach used by HamburgerMenu.

Output:
- PrivacyNotice.tsx — always-mounted, translate-y-based slide, backdrop unchanged
- TrademarkNotice.tsx — always-mounted, translate-y-based slide, delay-150 removed
- App.tsx — conditional guards removed, panels rendered unconditionally
</objective>
